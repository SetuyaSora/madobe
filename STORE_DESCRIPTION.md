# 窓辺 (Madobe) - ストア掲載用説明文 (日本語)

## キャッチコピー
いつものブラウザに、新しい景色を。

## 詳細説明
新しいタブを開くことは、部屋の窓を開けること。

『窓辺 (Madobe)』は、Chromeの新規タブを美しい動画で彩る、ミニマルな背景カスタマイズツールです。
お気に入りのループ動画を背景に設定し、すりガラスのようなグラスモルフィズムウィジェットを置けば、そこはあなただけのパーソナルスペースになります。

せわしない作業の合間に、ふと新規タブを開いて、心地よい「ゆらぎ」に目を留める。
デスクの上が少しだけ豊かになる、そんな新しいブラウジング体験をあなたのChromeへ。

---

◆ 心地よさと両立する、スマートな機能性

1. 動作は、空気のように軽く（省電力設計）
バックグラウンドでタブが隠れている間は、動画の再生を完全に一時停止します。CPU・GPUの消費を抑え、ノートPCのバッテリーにも極めて優しい設計です。

2. あなたの好きな動画を、そのまま背景に
設定パネルのファイル選択からお手持ちの動画ファイル（MP4/WebM）を選ぶだけで、ブラウザの内部ストレージ（IndexedDB）に安全に保存。次回起動時もローカルから自動で再生されます。

3. 浮遊する、すりガラスのウィジェット
デジタル/アナログ時計、カレンダー、ToDoリスト、RSSニュースフィード、Google検索窓を、48x24の緻密なグリッド上に配置。
ウィジェットを「長押し（ロングプレス）」することで、ぷるぷると震える編集モードが起動し、このモード中のみ、自由な配置変更やサイズ調整、滑らかな位置入れ替えが行えます。

4. 画面を隠さない、ホバードロワー
よく使うサイトへのショートカットは、画面の左端にマウスを合わせたときだけ滑らかにスライドインする「ホバードロワー」に格納。動画の美しさを邪魔しません。

5. 理想の文字をまとう
お好みの Google Fonts から、画面全体のフォントを動的に適用可能。文字の太さや質感まで、あなたの美意識に合わせられます。

* 安心の設計: 設定はJSONとしてエクスポート・インポートでき、バックアップも簡単です。
