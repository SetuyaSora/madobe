# 企画書：Chrome Wallpaper (仮称)

## 1. プロジェクト概要
本プロジェクトは、Google Chromeの新規タブ（ホーム画面）の背景に、ユーザーが指定した任意の動画を設定・再生できる軽量なChrome拡張機能です。Wallpaper Engineのように動的で美しい背景をChrome上で実現しつつ、ブラウザの動作を妨げない「軽さ」と「省電力」を両立させることを目指します。

## 2. ターゲット・利用シーン
- Chromeの新規タブ画面（ホーム画面）を自分好みにカスタマイズしたいユーザー
- Wallpaper Engineなどの動的壁紙が好きだが、ブラウザ単体で完結させたいユーザー
- PCのスペックに優しく、軽量かつスタイリッシュなブラウザ環境を構築したいユーザー

## 3. 主要機能（要件の具体化）
1. **背景動画再生機能**
   - Chromeの新規タブを開いた際に、背景で動画が自動再生（ループ・ループ制限設定）されること。
   - ユーザーが任意の動画ファイル（MP4, WebM, Oggなど）をアップロード（ドラッグ＆ドロップ）して設定できること。
   - （オプション）外部動画URL（YouTubeやDirect Link等）の指定。※パフォーマンスとプライバシーの観点から、まずはローカルファイル指定を優先。
2. **パフォーマンス・軽量化（「軽いこと」の実現）**
   - **非アクティブ時の自動一時停止**：新規タブがバックグラウンドに回った際（別のタブを見ている時、ウィンドウが最小化された時など）は、動画の再生を完全に一時停止し、CPU/GPUおよびメモリの消費をゼロに抑える。
   - **軽量なフレームワーク選定**：余計なランタイムを持たないVanilla HTML / Vanilla CSS / Vanilla JSのみで構築し、読み込み速度とメモリ使用量を最小限にする。
   - **ハードウェアアクセラレーションの活用**：CSSの `will-change` や適切なビデオデコーダーの活用により、描画負荷を低減する。
3. **新規タブとしての実用性（モダンなUI）**
   - **検索バー**：Google検索ができるシンプルな検索窓の設置。
   - **クイックリンク（ショートカット）**：よく使うサイトへのリンクを登録・編集・削除できる機能。
   - **設定パネル**：動画の変更、音量の調整（デフォルトはミュート）、再生速度、不透明度、検索バーやリンクの表示/非表示を切り替える設定UI（モダンなグラスモルフィズムデザイン）。
   - **フォントカスタマイズ機能**：設定画面から画面全体のベースフォントを自由に変更できる機能。厳選された美しい日本語・英語フォントプリセット（Inter, Noto Sans JP, Noto Serif JP, M PLUS Rounded等）から選べるほか、任意の Google Fonts の名前を入力して動的インポートできるハイブリッド方式を採用。
   - **ToDoウィジェット**：チェックボックスによる完了切り替えが可能なリスト形式のタスク管理ウィジェット。タスク全体の達成率を示すグラデーション進捗バー、および「すべて」「未完了」「完了済み」の絞り込みができるステータスフィルター機能を搭載。

## 4. 技術スタック
- **仕様**: Chrome Extension Manifest V3
- **フロントエンド**: HTML5, Vanilla CSS, Vanilla JavaScript (ES6+)
- **アイコンライブラリ**: Lucide Icons (SVG / 商用利用可能なフリーアイコン)
- **データ保存**: Chrome Extension `storage.local` または Web `IndexedDB`（動画ファイルをローカルに保存して次回以降も自動読み込みするため）

## 5. 公開に向けた決定事項と調整
- **動画の保存方法**: IndexedDB を用いたBlob形式の保存を採用。巨大な動画の登録を防ぐため、UI上で容量目安（50MB以下推奨）を提示する。
- **デフォルト動画の選定**: 完全に著作権フリー（CC0）で軽量なMDNデモ用動画（`assets/default_bg.mp4`、約1.1MB）をプリセットとして同梱。
- **初期ウィジェットの構成**: インストール直後の画面に「デジタル時計」「検索バー」「カレンダー」「ToDoリスト（サンプルタスク付き）」をあらかじめ美しく配置し、即座に実用性を体験できる仕様とした。
- **ビルド・配布の最適化**: 公開パッケージに必要なファイルのみを自動抽出してZIP化する `build.py` スクリプトを追加し、開発用ファイルや大容量画像を自動で除外する構成を整備。
