# 仕様書：Chrome Wallpaper

## 1. システム構成・拡張機能構造
本機能は、Chromeの新規タブ（New Tab）ページをカスタムHTMLにオーバーライドするChrome拡張機能（Manifest V3）として構築します。

### ディレクトリ構成
```text
chrome-wallpaper/
├── manifest.json         # 拡張機能のマニフェストファイル
├── newtab.html           # 新規タブのメインHTML
├── newtab.css            # スタイリング（グラスモルフィズム、レスポンシブ）
├── newtab.js             # メインロジック（動画制御、設定管理、検索、ショートカット）
├── assets/
│   ├── icon16.png        # 拡張機能用アイコン (16x16)
│   ├── icon48.png        # 拡張機能用アイコン (48x48)
│   ├── icon128.png       # 拡張機能用アイコン (128x128)
│   └── default_bg.mp4    # 内蔵の軽量サンプル動画 (デフォルト壁紙)
├── 企画書.md             # プロジェクト企画書
└── 仕様書.md             # 本仕様書
```

## 2. 機能要件

### 2.1. 背景動画再生機能
- **ビデオ要素 (`<video>`)**:
  - 画面全体にフィット（`object-fit: cover`）させ、背面（`z-index: -2`）に配置する。
  - ループ再生 (`loop`)、自動再生 (`autoplay`)、初期ミュート (`muted`) を有効にする。
- **壁紙ソース切り替え**:
  - **プリセット（デフォルト）**: 拡張機能内に同梱された `assets/default_bg.mp4` を再生。
  - **外部URL**: ユーザーが指定した動画URL（例: HTTPS経由のMP4/WebM）を再生。URLは `chrome.storage.local` に保存し、次回起動時も自動ロードする。
  - **ローカルファイル**: 
    - ユーザーのPCからファイルを選択して再生。
    - **永続化 (IndexedDB)**: 選択された動画データ（Blobオブジェクト）は、ブラウザ内部のデータベースである `IndexedDB` に一意のキーで永続保存される。これにより、新規タブの再読み込みやブラウザ再起動後もリセットされず、自動的に再生が再開される。
    - **容量とパフォーマンス**: 巨大な動画ファイルは読み込み遅延やブラウザの容量上限に達する恐れがあるため、「容量目安 50MB 以下の軽量動画を推奨」する旨をUIに明記する。
    - **ディスク容量の最適化**: ユーザーが壁紙ソースを「デフォルト」または「外部URL」に切り替えた場合、不要なディスクスペース消費を防ぐため、IndexedDB から保存済みの動画データを自動的に削除（クリア）する。

### 2.2. パフォーマンス・軽量化要件 (「軽いこと」の実現)
- **タブ非アクティブ時の再生停止**:
  - `document.visibilityState === 'hidden'`（別タブの閲覧中、またはウィンドウ最小化時）を検知し、動画の再生を一時停止（`video.pause()`）する。
  - タブが再度アクティブになったら再生を再開（`video.play()`）する。
- **CSS最適化**:
  - ビデオ要素に対して `will-change: transform` または `transform: translateZ(0)` を適用し、GPUアクセラレーションを有効化する。
- **軽量アセット**:
  - 内蔵するデフォルト動画（`default_bg.mp4`）は、高圧縮でデコード負荷の低いMP4/WebMフォーマットを使用する。

### 2.3. 新規タブ画面 UI・コンテンツ
- **検索バー**:
  - 画面中央に配置。Google検索に対応。
  - 入力してEnterキーを押すことで、Google検索結果ページへ遷移する。
- **クイックリンク (ショートカット)**:
  - 検索バーの下部によく使うサイトのアイコンを表示。
  - ユーザーが新しいリンクを追加、既存のリンクを編集・削除できるようにする。
  - リンクデータは `chrome.storage.local` に保存し永続化。
- **設定パネル (オーバーレイ)**:
  - 画面の右下に配置されたギアアイコン（SVG）をクリックすることで開閉。
  - グラスモルフィズムデザイン（背景ぼかし `backdrop-filter: blur`）を採用した、モダンで半透明なパネル。
  - **設定項目**:
    - 壁紙ソースの選択（デフォルト / 外部URL / ローカルファイル永続再生）
    - 外部URL入力欄（保存ボタン付き）
    - ローカルファイル選択ボタン
    - 音量コントロール（0%〜100%、スライダー）
    - 再生速度（0.5x, 1.0x, 1.5x, 2.0x、ラジオボタンまたはセレクトボックス）
    - 壁紙の不透明度（0%〜100%のスライダー、動画の上に重ねる暗色オーバーレイの透明度を調整し、検索バーなどの文字の視認性を確保する）
    - 検索バーの表示/非表示トグル

## 3. 非機能要件・デザインシステム
- **デザイン・美観**:
  - システム絵文字は使用せず、Lucide Icons（SVGパス）またはCSSによるモダンなアイコンを採用。
  - フォントは、OSの標準サンセリフフォントに加え、Google Fonts（Inter等）をインポートしてモダンな印象にする。
  - ダークモードを意識したスタイリッシュな半透明（グラスモルフィズム）デザイン。
- **レスポンシブ**:
  - さまざまな画面解像度（ノートPC、デスクトップ、4Kモニターなど）に対応。

## 4. 開発環境・API仕様
- **Chrome Extension Manifest**: `manifest.json` (Version 3)
- **パーミッション (`permissions`)**:
  - `storage` (設定情報およびショートカットリンクの保存用)
- **オーバーライド設定**:
  - `chrome_url_overrides` を使用し、`newtab` ページを `newtab.html` にマッピング。
